Package info
####################

ncplot was created by Robert Wilson at Plymouth Marine Laboratory (PML).


Bugs and issues
------------------

If you identify bugs or issues with the package please raise an issue at PML's Marine Systems Modelling group's GitHub page `here <https://github.com/pmlmodelling/ncplot/issues>`__ or contact nctoolkit's creator at rwi@pml.ac.uk.


Contributions welcome
------------------
The package is new. It should be able to handle most types of NetCDF file, but there will still be times when errors are thrown.  If packages users would like to contribute, suggest new features or point to data the package does not work with they are welcome to raise and issue at the package's GitHub page or contact me.
